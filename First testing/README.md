# Brain Controlled Command

Brain-Computer Interface (BCI) system that leverages electroencephalography (EEG) signals to control external devices in real-time. This project demonstrates how EEG signals can be classified into different mental states (attentive and relaxed) to control a car racing game without physical input devices.
